Demo 7 : 좌석 이름 변경

  • 좌석 이름을 숫자와 알파벳 사이에서 선택할 수 있음
  • 줄 이름은 data-rowname="alphabet" 또는 data-rowname="number" 중 선택 가능(기본값 number)
  • 칸 이름은 data-colname="alphabet" 또는 data-colname="number" 중 선택 가능(기본값 number)
  • 예제는 줄은 알파벳, 칸은 숫자로 설정되어 있음
  • 알파벳일 경우 무조건 대문자 A부터 시작, 숫자일 경우 무조건 1부터 시작(변경불가)
현재 선택된 좌석 수 : 0
